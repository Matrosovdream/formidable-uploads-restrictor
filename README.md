# Formidable – Restrict Entry Files

Protect all files under `/wp-content/uploads/formidable/` by requiring users to be logged in and belong to specific roles.

## Features
- Keeps the same public URLs.
- Blocks guests and unauthorized users.
- Adds Formidable → Restrict entry files setting.

## Install
1. Upload `formidable-restrict-uploads` to `/wp-content/plugins/`.
2. Activate the plugin.
3. Set allowed roles under **Formidable → Restrict entry files**.
